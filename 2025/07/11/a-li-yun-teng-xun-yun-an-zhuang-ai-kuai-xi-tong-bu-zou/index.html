阿里云/腾讯云 安装爱快软路由系统

在阿里云或腾讯云上安装爱快（iKuai）软路由系统，可以将云服务器变成功能强大的路由器，实现多线负载均衡、行为管理、流量控制等功能。整个过程分为六个步骤。

前提条件

  • 已购买阿里云或腾讯云服务器
  • 操作系统选择 Ubuntu（版本任意，推荐 20.04/22.04）
  • 能通过控制台使用 VNC 远程登录

第一步：设置交换机/子网（关键）

这一步非常重要！云厂商默认分配的私有地址段过大（如 /16 或 /20），而爱快在设置 WAN 口时无法指定子网掩码，必须先缩小地址段。

阿里云操作

  1. 进入控制台 → 点击 ECS 实例 → 左下角 专有网络 VPC
  2. 新建交换机，设置合适的子网，例如 172.18.1.0/24
  3. 停止实例（必须先停机）
  4. 点击实例 → 修改私有 IP → 选择刚创建的交换机 → 指定 IP（如 172.18.1.1）
  5. 启动实例

如果一次修改不成功，可能需要多次保存重试。

腾讯云操作

  1. 进入控制台 → 云产品 → 私有网络
  2. 点击 子网 → 新建子网，例如 172.16.16.0/24
  3. 点击实例名 → 进入 弹性网卡 设置 → 点击主网卡名
  4. 修改私网 IP 为新子网内的地址（如 172.16.16.3）

第二步：记录 IP 地址和网关

启动 Ubuntu 系统后，登录并查看网络信息，抄下来备用。

阿里云

用 root 账号登录：

ip addr    # 查看 IP 地址
ip route   # 查看网关

例如查到：IP 172.18.1.1，网关 172.18.1.253。

腾讯云

用 ubuntu 账号登录，再切换 root：

sudo su -
ip addr
ip route

例如查到：IP 172.16.16.3，网关 172.16.16.1。

腾讯云建议全程使用 VNC 方式操作。

第三步：设置安全组

在云控制台配置安全组，放行以下端口：

  • 80/443：爱快 Web 管理界面
  • 其他端口：根据实际需求放行（如 VPN 端口等）

第四步：下载爱快 ISO

登录 Ubuntu 系统，下载爱快安装镜像。

32 位版本（云主机配置有限时推荐）：

wget https://www.ikuai8.com/download.php?n=/3.x/iso/iKuai8_x32_3.7.14_Build202408011011.iso -O ik.iso

64 位版本：

wget https://www.ikuai8.com/download.php?n=/3.x/iso/iKuai8_x64_3.7.14_Build202408011011.iso -O ik.iso

可前往 爱快官网 获取最新版本链接。

第五步：挂载 ISO 并引导安装

方法一：挂载启动文件（推荐）

# 挂载 ISO
sudo mount ik.iso /mnt

# 复制启动文件到 boot 分区
sudo cp -rpf /mnt/boot /

# 重启
sudo reboot

重启后系统会自动进入爱快安装界面。

方法二：进入 GRUB 手动引导

# 下载完成后重启
reboot

阿里云：重启时按住 ESC 键，直到进入 GRUB 界面，按 c 进入命令行模式。

腾讯云：VNC 不太好操作 GRUB，建议提前删除 GRUB 配置文件强制停留：

sudo rm /boot/grub/grub.cfg
sudo reboot

进入 GRUB 命令行后，输入引导命令启动安装程序。

安装过程中如果出现报错，可以忽略，不影响最终安装。

第六步：配置爱快网络

安装完成后进入爱快控制台：

  1. 选择 1 进入网络接口设置
  2. del lan1 — 删除默认的 LAN 口（云服务器只有一个网口，用不到 LAN）
  3. 将唯一的网口设为 WAN：set wan1 eth0
  4. 选择 2 → 1，设置 WAN 口地址，填入 第二步记录的 IP 和网关
  5. 选择 o（其他选项）→ 2，开启外网 Web 访问

完成！现在可以通过浏览器访问爱快管理界面了：

http://你的公网IP

默认账号：admin，默认密码：admin

常见问题

Q: 安装时报错怎么办？

安装界面出现一些红色报错是正常的，忽略即可，不影响使用。

Q: 修改私有 IP 时提示失败？

可能需要多次尝试保存，或者确认实例已完全停止。

Q: 重启后进不去爱快？

检查启动文件是否正确复制到了 /boot 目录，或者尝试通过 VNC 手动进入 GRUB 引导。

Q: 忘记了内网 IP 怎么办？

通过云控制台的 VNC 登录爱快终端，查看 WAN 口配置。
