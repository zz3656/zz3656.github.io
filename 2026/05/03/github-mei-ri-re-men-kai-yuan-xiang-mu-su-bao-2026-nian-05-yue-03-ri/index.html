1. freeCodeCamp/freeCodeCamp

freeCodeCamp 是全球最大的免费编程学习平台，提供从 Web 开发到机器学习的完整课程体系。涵盖 JavaScript、React、D3.js、Node.js 等技术栈，学完可获得免费认证证书。

  • 核心功能：交互式编程课程、项目实战、免费认证、社区论坛
  • 适用场景：零基础入门编程、转行学开发、准备技术面试
  • 快速上手：访问 freecodecamp.org 注册即可开始学习

2. public-apis/public-apis

一个由社区维护的免费公共 API 列表，涵盖天气、金融、音乐、AI、游戏等数十个领域。无论你做什么项目，大概率能在这里找到合适的 API。

  • 核心功能：分类清晰的免费 API 索引、社区持续更新
  • 适用场景：开发练手项目需要数据源、快速原型验证、学习 API 调用
  • 快速上手：直接浏览 GitHub 仓库 按分类查找

3. openclaw/openclaw

一个跨平台（Windows/macOS/Linux/iOS/Android）的个人 AI 助手，主打”own your data”——所有数据完全由用户掌控。可以连接多种大语言模型，定制个人工作流。

  • 核心功能：多平台客户端、多模型支持、本地优先、插件扩展
  • 适用场景：构建私人 AI 助手、替代 ChatGPT 等商业产品、隐私敏感场景
  • 快速上手：git clone https://github.com/openclaw/openclaw && cd openclaw && npm install

4. nilbuild/developer-roadmap

roadmap.sh 的开源仓库，提供前端、后端、DevOps、AI、区块链等数十条开发者成长路线图。每条路线图都是交互式的，标注了每个阶段需要掌握的技能和推荐资源。

  • 核心功能：交互式路线图、最佳实践指南、技术问答
  • 适用场景：制定学习计划、技术选型参考、团队技能评估
  • 快速上手：访问 roadmap.sh 选择感兴趣的方向

5. vinta/awesome-python

Python 生态最全面的资源精选列表，收录了框架、库、工具、学习资源等数百个项目。被收录的项目都是社区公认的高质量选择。

  • 核心功能：按类别组织的 Python 资源索引
  • 适用场景：Python 新手寻找靠谱工具、技术选型对比、发现新库
  • 快速上手：浏览 GitHub 仓库 按需查阅

6. awesome-selfhosted/awesome-selfhosted

自托管（Self-Hosted）软件的终极列表。收录了数百个可以部署在自己服务器上的免费网络服务和 Web 应用，涵盖博客、网盘、邮件、项目管理等几乎所有品类。

  • 核心功能：分类齐全的自托管方案索引、开源许可证标注
  • 适用场景：搭建私有云服务、替代 SaaS 产品节省费用、数据隐私合规
  • 快速上手：浏览 仓库列表 找到需要的类别

7. facebook/react

Meta（原 Facebook）出品的前端 UI 库，全球使用最广泛的前端框架之一。采用声明式编程范式和组件化架构，配合 React Native 还可开发移动端应用。

  • 核心功能：虚拟 DOM、JSX、Hooks、并发渲染、Server Components
  • 适用场景：构建大型 Web 应用、跨平台移动开发（React Native）
  • 快速上手：npx create-react-app my-app && cd my-app && npm start

8. torvalds/linux

Linus Torvalds 维护的 Linux 内核官方仓库。Linux 内核是所有 Linux 操作系统的核心，管理硬件、系统资源，为所有上层软件提供基础服务。

  • 核心功能：操作系统内核、硬件驱动管理、进程调度、内存管理
  • 适用场景：内核开发学习、驱动开发、操作系统研究
  • 快速上手：git clone https://github.com/torvalds/linux.git

9. tensorflow/tensorflow

Google 出品的开源机器学习框架，支持从训练到部署的完整 ML 工作流。提供 Python、C++、Java 等多语言 API，支持 CPU/GPU/TPU 加速。

  • 核心功能：神经网络构建与训练、模型部署（TF Serving/TF Lite）、可视化（TensorBoard）
  • 适用场景：深度学习研究与生产部署、移动端/嵌入式 AI 推理
  • 快速上手：pip install tensorflow

10. ultraworkers/claw-code

用 Rust 构建的极速代码编辑器，号称史上最快突破 10 万星标的开源项目。基于 oh-my-codex 引擎，主打极致性能和智能编码体验。

  • 核心功能：Rust 高性能编辑器、智能补全、多语言支持
  • 适用场景：追求极致启动速度的轻量级编码、Rust 开发环境
  • 快速上手：参考仓库中的 USAGE.md

1. nexu-io/open-design

Anthropic Claude Design 的开源替代品，支持本地优先运行。内置 31 个可组合技能和 72 套品牌级设计系统，可以生成网页、桌面、移动端原型、幻灯片、图片和视频。支持 Claude Code、Codex、Cursor、Gemini 等 12 种编码 Agent CLI 自动检测。

  • 核心功能：AI 驱动设计生成、多端原型输出、HTML/PDF/PPTX/MP4 导出、沙盒预览
  • 适用场景：快速 UI 原型设计、品牌视觉生成、Vibe Coding 工作流
  • 快速上手：git clone https://github.com/nexu-io/open-design && cd open-design && npm install

2. cursor/cookbook

Cursor 官方推出的编码 Agent 实战手册。包含 Cursor SDK（TypeScript API）的各类使用示例，可以让你在自己的应用和脚本中调用 Cursor 的编码 Agent 能力。

  • 核心功能：Cursor SDK 使用示例、Agent 工作流模板、流式事件处理
  • 适用场景：将 Cursor Agent 能力集成到自己的工具链、自动化编码流程
  • 快速上手：git clone https://github.com/cursor/cookbook

3. theori-io/copy-fail-CVE-2026-31431

一个 Linux 内核安全漏洞（CVE-2026-31431）的研究项目和 PoC。涉及 copy 操作中的失败处理逻辑，影响了 Ubuntu 24.04、Amazon Linux 2023、RHEL 10.1、SUSE 16 等主流发行版。

  • 核心功能：漏洞 PoC、受影响版本清单、技术分析报告
  • 适用场景：Linux 安全研究、漏洞分析学习、企业安全审计
  • 快速上手：阅读 技术分析报告

4. denuitt1/mhr-cfw

一个域名前置（Domain Fronting）中继工具，通过 Google Apps Script 转发流量到 Cloudflare Workers，用于绕过深度包检测（DPI）。仅供教育和研究用途。

  • 核心功能：GAS + CF Workers 流量中继、域名前置、DPI 规避
  • 适用场景：网络 censorship 研究、安全测试、渗透测试
  • 快速上手：参考仓库 README 配置 GAS 和 CF Workers

5. willchen96/mike

一个开源的 AI 法律平台，包含 Next.js 前端和 Express + Supabase 后端。提供文档处理、法律咨询等 AI 驱动的法律服务功能。

  • 核心功能：AI 法律文档分析、Supabase 数据管理、文档迁移系统
  • 适用场景：法律科技（LegalTech）创业、法律文档自动化、AI + 法律研究
  • 快速上手：
git clone https://github.com/willchen96/mike
cd mike
npm install --prefix backend && npm install --prefix frontend

6. darrylmorley/whatcable

一个 macOS 菜单栏小工具，用简洁的英文告诉你每根 USB-C 线缆的实际能力——是否支持快充、数据传输速度、Thunderbolt 支持等。解决了”看起来一样但能力天差地别”的 USB-C 线缆混乱问题。

  • 核心功能：USB-C 线缆能力识别、充电慢原因分析、Thunderbolt 支持检测
  • 适用场景：Mac 用户排查充电问题、确认线缆是否支持高速数据传输
  • 快速上手：从 GitHub Releases 下载安装

7. DanOps-1/Gpt-Agreement-Payment

ChatGPT Plus/Team/Pro 订阅协议的端到端重放工具集，附带从零实现的 hCaptcha 视觉求解器。定位为反欺诈机制的实证研究项目。

  • 核心功能：订阅协议重放、hCaptcha 求解器、反欺诈数据采集
  • 适用场景：网络安全研究、CAPTCHA 机制分析、Bug Bounty
  • 注意：仅供安全研究和教育用途

8. b-nnett/codex-plusplus

Codex 桌面应用的增强插件系统（Tweak System），可以注入自定义功能、修复 UI 问题、添加扩展管理器，无需重新编译应用本身。已确认在 macOS 和 Windows 上可用。

  • 核心功能：Codex 应用增强、自定义功能注入、Tweak 管理器
  • 适用场景：定制化 Codex 使用体验、修复官方未解决的 UI 问题
  • 快速上手：参考仓库 README 安装 Tweak 系统

9. GENEXIS-AI/chromex

一个由 Codex 驱动的 Chrome 浏览器侧边栏 AI 助手。支持页面上下文理解、标签页管理、语音交互和图片处理工作流。

  • 核心功能：页面上下文 AI 问答、标签页智能管理、语音输入、图片工作流
  • 适用场景：浏览器内 AI 辅助阅读、网页内容提取与总结
  • 快速上手：从 Chrome Web Store 安装（详见仓库说明）

10. t8y2/dbx

一个轻量级、跨平台的数据库管理工具（GUI 客户端）。基于 Tauri + Vue 构建，支持 MySQL、PostgreSQL、SQLite、Redis、MongoDB、DuckDB、ClickHouse、SQL Server 等主流数据库。

  • 核心功能：多数据库连接管理、SQL 查询编辑器、跨平台（Tauri）
  • 适用场景：替代 DBeaver/Navicat 等重量级工具、日常数据库管理运维
  • 快速上手：从 GitHub Releases 下载对应平台安装包
