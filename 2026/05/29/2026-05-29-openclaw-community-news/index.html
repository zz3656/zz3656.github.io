📌 什么是 OpenClaw？

OpenClaw 是一个免费、开源的 AI Agent 框架，可以把大语言模型变成能在你的电脑上执行真实任务的助手。最早于 2025 年 11 月以 Clawdbot 的名字发布，随后改名为 OpenClaw，迅速积累了 145,000+ GitHub Stars 和 20,000+ Forks，是增长最快的开源 AI 项目之一。

🌐 OpenClaw 官方介绍

Showcases, skills directory, ready-made templates, and curated news — everything the OpenClaw community needs, in one place.
OpenClaw is a free, open-source AI agent framework that turns large language models into hands-on assistants capable of executing real tasks on your computer. Originally released under the name Clawdbot in November 2025, it was created by developer Peter Steinberger and has since attracted over 145,000 GitHub stars and 20,000 forks — making it one of the fastest-growing open-source AI projects in history.
Unlike cloud-only AI chatbots, the agent runs entirely on your local machine. It connects to any large language model — Claude, GPT, DeepSeek, or self-hosted open-weight models — and uses messaging platforms like Telegram, Signal, and WhatsApp as its primary interface. That means you can text your agent from your phone and have it manage emails, update calendars, scrape the web, generate reports, or orchestrate multi-step workflows while you focus on higher-level work.
The OpenClaw ecosystem revolves around AgentSkills — modular plugins that extend what your agent can do. Over 100 community-built skills are already available, covering everything from news aggregation and semantic memory to browser automation and CI/CD monitoring. Because the framework is model-agnostic and privacy-focused, your data stays on your hardware and your API keys stay under your control.
openclaw-ai.dev exists to serve the global OpenClaw community. We are an independent hub — not affiliated with the official project — dedicated to curating the best agent showcases, maintaining a searchable skills directory, offering one-click templates, and publishing weekly news so that every user, from beginner to power user, can get the most out of their AI agent setup.

Frequently asked questions
OpenClaw is a free, open-source AI agent framework (formerly known as Clawdbot and Moltbot) that runs entirely on your local machine. It connects to large language models such as Claude, GPT, or DeepSeek and automates everyday tasks through messaging platforms like Telegram, Signal, and WhatsApp. With over 145,000 GitHub stars, the project is one of the most popular open-source AI efforts in the world.
openclaw-ai.dev is an independent community hub built around the OpenClaw ecosystem. We curate the best agent showcases from around the world, maintain a searchable skills directory with ratings and install counts, provide production-ready templates, and publish a weekly newsletter covering the latest news, tutorials, and security advisories. We are not affiliated with the official project.
Browse our Skills Directory, find the skill you need, and copy the one-line install command. Most skills can be added via the CLI — for example: openclaw install news-aggregator. Alternatively, you can paste the skill’s JSON block directly into your agent configuration file and restart the service.
We offer both free and Pro templates. Free templates cover everyday use cases such as customer support bots and automated daily reports. Pro templates target more advanced scenarios — multi-channel SEO monitoring, cross-platform social media management, and research pipelines — and include priority updates and dedicated support.
Click the ‘Submit a Skill’ button in our Skills Directory section or reply to any edition of our weekly newsletter. We review every community submission and feature the best agents and skills on the homepage. Developers can also list premium skills with a promoted placement through our sponsor program.
Because the framework is open-source and runs locally, you retain full control over your data and API keys. That said, agents can access sensitive services such as email, calendars, and messaging apps, so you should always review skill configurations before installing them. Never expose your agent gateway to the public internet, and keep your instance updated to benefit from the latest security patches.
