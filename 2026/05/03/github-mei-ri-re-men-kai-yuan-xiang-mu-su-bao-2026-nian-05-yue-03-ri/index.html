  • 简介：GitHub 上最具影响力的精选资源合集，涵盖编程、AI、DevOps、设计等几乎所有技术领域。
  • 核心功能：社区驱动的 Awesome List 总目录，汇聚数百个分类子列表，每个子列表由领域专家维护，持续更新高质量开源项目、学习资源和工具推荐。
  • 使用场景：快速了解某个技术领域的生态全貌；寻找特定方向的优质开源工具；技术选型时做横向对比参考。
  • 快速上手：直接浏览 awesome.md，按分类索引找到感兴趣的主题。

2. freeCodeCamp/freeCodeCamp

GitHub：https://github.com/freeCodeCamp/freeCodeCamp

  • 简介：全球最大的免费编程学习平台，帮助超过 10 万人成功转型进入科技行业。
  • 核心功能：提供全栈 Web 开发与机器学习的完整课程体系，包含数千个交互式编程挑战；支持响应式 Web 设计、JavaScript 算法、前端库、数据可视化、API 等认证路径；完全免费、自主节奏学习。
  • 使用场景：编程零基础入门学习；准备开发者面试刷题；系统学习全栈开发技能。
  • 快速上手：访问 freecodecamp.org 注册账号即可开始学习，无需安装任何软件。

3. public-apis/public-apis

  • 简介：社区精心整理的免费公开 API 列表，堪称开发者的 API 宝库。
  • 核心功能：按类别（动物、动漫、反恶意软件、艺术、认证等）组织上千个免费 API，每个条目标注认证方式、HTTPS 支持和 CORS 策略，方便开发者快速筛选适合的 API。
  • 使用场景：开发个人项目或原型时快速找到合适的 API；学习 API 调用和数据集成；寻找特定领域（天气、金融、地图等）的数据源。
  • 快速上手：直接查阅 README 中的分类列表，找到所需 API 即可按文档调用。

  • 使用场景：搭建个人专属 AI 助手连接常用聊天工具；本地部署 AI 保护隐私数据；跨平台统一 AI 交互入口。
