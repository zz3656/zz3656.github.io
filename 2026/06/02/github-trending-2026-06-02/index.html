数据来源：GitHub API，统计周期为最近 7~14 天内创建的新建仓库，分别按 星标总数 和 每日增速 两个维度排名。

📈 Part 1：本周 star 总量最高的 10 个新项目

按仓库星标总数排序，统计周期内创建的所有新仓库中，星标总数最多的 Top 10。

1. odysseus — 自托管 AI 工作空间

  • ⭐ 总星标：17,162（创建仅 2 天，日均增速 8,581 ⭐）
  • 🔗 仓库：pewdiepie-archdaemon/odysseus
  • 语言：JavaScript

项目介绍：一个完全自托管的 AI 工作空间，提供类似 Next.js/Web 应用的交互界面，支持多 AI 模型接入、文件管理、代码编辑等能力，数据完全私有，适合对隐私有要求的团队。

使用场景：个人/团队搭建私有 AI 开发环境；企业内网部署 AI 助手；需要数据不离线的开发工作流。

2. bumblebee — 软件供应链风险扫描器

  • ⭐ 总星标：4,092（13 天，日均 314 ⭐）
  • 🔗 仓库：perplexityai/bumblebee
  • 语言：Go

项目介绍：Perplexity AI 开源的只读端点扫描器，用于检测本地磁盘上已安装的包、扩展和开发工具是否存在已知供应链安全漏洞（如依赖混淆、恶意包等），帮助开发者排查暴露面。

使用场景：CI/CD 流水线安全审计；开发者本地环境自查；企业安全团队批量排查开发工具风险。

3. PilotDeck — 任务导向 AI Agent 生产平台

  • ⭐ 总星标：2,699（11 天，日均 245 ⭐）
  • 🔗 仓库：OpenBMB/PilotDeck
  • 语言：TypeScript

项目介绍：面向任务的 AI Agent 生产力平台，支持多步骤任务规划、工具调用和工作流编排，帮助用户将复杂任务拆解为可自动执行的 Agent 流水线。

使用场景：自动化办公流程（表单填写、报告生成）；跨工具数据处理；个人/团队任务管理自动化。

4. OpenLogi — Logitech Options+ 开源替代

  • ⭐ 总星标：2,630（9 天，日均 292 ⭐）
  • 🔗 仓库：AprilNEA/OpenLogi
  • 语言：Rust

项目介绍：原生、本地优先的 Logitech 鼠标/键盘配置工具，使用 Rust 编写，通过 HID++ 协议直接与设备通信，无需登录、无遥测数据，完全离线运行。

使用场景：Logitech 高端鼠标用户自定义按键/DPI/SmartShift；注重隐私、不希望安装官方驱动联网的用户；Linux/macOS 用户（官方软件支持较差）的替代方案。

5. 9arm-skills — AI 编程技巧集合

  • ⭐ 总星标：2,591（13 天，日均 199 ⭐）
  • 🔗 仓库：thananon/9arm-skills
  • 语言：Shell

项目介绍：收录了大量 AI 编程技巧和 Claude Code / Codex 使用经验的技能集合，帮助开发者快速上手 AI 辅助编码，提升使用效率。

使用场景：新手开发者学习 AI 辅助编程；团队内部沉淀 AI 使用规范；快速掌握主流 Agent 工具的实战技巧。

6. guizang-social-card-skill — 小红书/公众号配图生成

  • ⭐ 总星标：2,388（6 天，日均 398 ⭐）
  • 🔗 仓库：op7418/guizang-social-card-skill
  • 语言：HTML

项目介绍：Claude Code / Codex Skill，支持生成小红书图文和微信公众号封面，支持 28 种版式、10 种主题风格，单 HTML 文件即可输出 PNG 图片，瑞士编辑设计系统驱动视觉风格。

使用场景：内容创作者批量生成小红书配图；自媒体运营快速制作公众号封面；AI 辅助内容生产流水线。

7. gsd-core — Git Ship Done

  • ⭐ 总星标：2,124（11 天，日均 193 ⭐）
  • 🔗 仓库：open-gsd/gsd-core
  • 语言：JavaScript

项目介绍：快速将代码从想法到产出的开发工作流工具链核心，聚焦于减少从 commit 到部署的摩擦，提供模板化、标准化的高速交付体验。

使用场景：初创团队快速迭代开发流程；开源项目维护者标准化 PR/CD 流程；个人开发者简化发布步骤。

8. smartNode — 智能节点编排

  • ⭐ 总星标：2,010（12 天，日均 167 ⭐）
  • 🔗 仓库：Tong89/smartNode
  • 语言：Python

项目介绍：轻量级智能节点自动化编排工具，支持可视化流程编排、插件扩展和多场景调度，适合构建自动化工作流和 AI 任务管道。

使用场景：自动化运维脚本编排；AI 任务流水线搭建；IoT 设备状态监控与自动化响应。

9. ian-xiaohei-illustrations — 中文小黑怪诞配图

  • ⭐ 总星标：1,609（6 天，日均 268 ⭐）
  • 🔗 仓库：helloianneo/ian-xiaohei-illustrations
  • 语言：—

项目介绍：专为中文内容设计的怪诞风格正文配图 Skill，16:9 白底手绘风格，少量红橙蓝批注点缀，适配 Claude Code / Codex，直接生成配图 Prompt。

使用场景：技术博客/教程文章配图；AI 辅助内容创作；中文自媒体独特风格配图生成。

10. kimi-code — 下一代 Agent 起点

  • ⭐ 总星标：1,545（11 天，日均 140 ⭐）
  • 🔗 仓库：MoonshotAI/kimi-code
  • 语言：TypeScript

项目介绍：Moonshot AI 出品的下一代 AI 编程助手起点项目，提供 Agent 化代码开发框架和工具链集成，是 Kimi 在编程场景的核心技术探索。

使用场景：AI 原生开发工具链构建；企业级代码开发 Agent 集成；AI 编程教育与研究。

OpenLogi 和 bumblebee 分属工具类和安全类，显示开源社区对「隐私优先」和「供应链安全」两个议题的持续关注。
