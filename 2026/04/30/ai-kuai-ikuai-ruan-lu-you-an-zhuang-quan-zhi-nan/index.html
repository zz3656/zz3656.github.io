什么是爱快(iKuai)软路由？

爱快(iKuai)是由全讯汇聚网络科技（北京）有限公司开发的商业级软路由系统，提供专业的网络解决方案。它支持多WAN接入、DPI流控、行为管理、VPN组网、无线AC管理等企业级功能，同时提供免费版本供个人用户使用。

爱快软路由的核心优势：

  • 免费且功能强大：免费版已涵盖大部分家用和小型企业需求
  • DPI深度数据包检测：精准识别上千种应用协议，实现精细化流量管控
  • 多线负载均衡：支持多条宽带线路叠加和智能切换
  • Web管理界面：直观易用的中文管理后台
  • 多平台安装：支持实体机、虚拟机、云主机等多种部署方式

官方网站：https://www.ikuai8.com

一、安装前的准备工作

1. 固件下载

前往爱快官网下载页面获取安装镜像：https://www.ikuai8.com/download

爱快提供以下格式的安装文件：

文件格式 适用场景
ISO 光盘安装、U盘引导安装、虚拟机安装
IMG 写盘工具直接写入硬盘/U盘
GHOST GHOST全盘恢复安装

2. 重要提示

注意：爱快系统只能安装在一块磁盘里，请确保安装机器里只有一块磁盘！
如果有多块磁盘，安装前请暂时断开其他磁盘的数据线，避免误写。

3. 硬件要求

  • 最低配置：x86/x64处理器、512MB内存、1GB以上存储空间
  • 推荐配置：多核处理器、2GB以上内存、8GB以上SSD/电子盘
  • 至少两块网卡（一个WAN口 + 一个LAN口）

二、真实机安装方式

真实机安装是爱快最常见的部署方式，将系统直接安装在物理硬件上，性能损耗最小。

方式一：ISO U盘引导安装（推荐）

这是最常用的安装方式，操作简单，适合大部分场景。

所需工具：一个U盘（2GB以上）、老毛桃U盘启动制作工具、爱快ISO镜像

操作步骤：

  1. 下载老毛桃U盘启动制作工具（https://www.laomaotao.net/）
  2. 插入U盘，打开老毛桃，选择U盘盘符
  3. 点击”一键制作成USB启动盘”，注意这会格式化U盘，请提前备份
  4. 制作完成后，在U盘根目录创建一个名为 LMT 的文件夹
  5. 将爱快ISO文件放入 LMT 文件夹中
  6. 重启电脑，设置BIOS从U盘启动
  7. 选择”自定义安装ISO/IMG”，系统会搜索LMT目录下的ISO文件
  8. 选择爱快ISO文件，进入安装界面
  9. 选择安装选项，输入 y 确认安装
  10. 安装完成后自动重启，拔掉U盘

注意：不同版本老毛桃的ISO存放目录可能是 ISOS 而非 LMT，请注意区分。如果LMT文件夹不存在需要手动创建。

方式二：UEFI引导安装

适用于支持UEFI启动的新硬件，安装过程更加规范。

操作步骤：

  1. 准备一个普通U盘
  2. 下载爱快UEFI引导文件（EFI文件），解压后找到EFI文件夹
  3. 将EFI文件夹和爱快ISO镜像一起放到U盘根目录
  4. 插上U盘，开机按F2进入UEFI启动页面
  5. 选择 BOOT Manager
  6. 选择 EFI USB Device（U盘引导），进入镜像选择页面
  7. 选择U盘中的系统镜像
  8. 选择安装目标硬盘
  9. 如果之前硬盘有安装过爱快系统，会提示是否保留配置，根据需求选择
  10. 安装完成后重启，拔掉U盘即可

方式三：IMG写盘安装

适合对Linux命令行不陌生、希望快速安装的用户。

操作步骤：

  1. 从官网下载IMG版安装包和img写盘工具
  2. 用RAR解压出 ikuai8.img 镜像文件
  3. 使用PE系统或Windows下的img写盘工具对目标硬盘进行写入
  4. 选择硬盘时必须选择 Physical Disk（物理磁盘）
  5. 支持U盘、IDE硬盘、SATA硬盘、电子硬盘等介质
  6. 写盘完成后重启即可

警告：写盘操作会覆盖目标磁盘上的所有数据，请务必确认选择了正确的磁盘！

方式四：GHOST全盘恢复安装

适合熟悉GHOST工具的运维人员。

操作步骤：

  1. 下载爱快GHOST安装版文件
  2. 启动GHOST，选择 Local -> Disk -> From Image
  3. 选择已下载的iKuai GHOST镜像包
  4. 按Tab键选择 OK，回车确认
  5. 选择目标磁盘（Destination Drive），一般选择默认即可
  6. 选择 Yes 确认开始恢复
  7. 等待GHOST完成，选择 Reset Computer 重启

方式五：ISO光盘引导安装

适用于有光驱的老硬件。

操作步骤：

  1. 下载爱快ISO镜像文件，刻录到光盘
  2. 设置电脑从光盘启动
  3. 重启后出现安装界面，选择选项 1
  4. 输入 y 确认安装
  5. 安装完成后自动重启，取出光盘

三、安装后的初始配置

1. 网线连接

安装完成后，将网线接到网卡上，在控制台按回车键：

  • 如果LAN口显示”已连接”，说明网线接对了
  • 检查内网是否有IP地址和爱快默认IP冲突

2. 登录Web管理界面

在浏览器地址栏输入爱快默认管理地址：

http://192.168.1.1

默认登录凭据：

  • 用户名：admin
  • 密码：admin

重要：首次登录后请立即修改默认密码！

3. U盘容量变小问题

安装爱快后如果发现U盘容量变小（比如16GB的U盘只剩50MB），这是正常现象，因为爱快系统重新格式化了U盘的分区结构。

解决方法：

  1. 下载老毛桃工具（https://www.laomaotao.net/）
  2. 选择此U盘，执行”初始化U盘”操作
  3. 还原后U盘容量恢复正常

四、虚拟机安装

虚拟机安装适合测试、学习以及不想单独用一台物理机的场景。推荐使用VMware虚拟机。

4.1 虚拟机环境搭建

步骤：

  1. 安装VMware虚拟机软件
  2. 下载爱快ISO镜像
  3. 创建新的虚拟机，选择爱快ISO镜像
  4. 系统类型建议选择 Ubuntu 64位（重要！）
  5. 按提示完成虚拟机创建
  6. 开启虚拟机，选择安装选项 1，回车自动安装
  7. 安装成功后自动重启，进入中文控制台

关键提示：个人版VMware建议系统类型选择Ubuntu 64位，否则默认网卡类型是AMD，性能很差。
如果已经安装好了，可以编辑虚拟机目录下的 .vmx 文件（关机状态），添加以下内容将网卡类型改为e1000：

ethernet0.virtualDev = "e1000"
ethernet1.virtualDev = "e1000"

重启后在”系统设置 -> 硬件信息”中确认网卡是否显示为 Intel Corporation 82545EM。

4.2 场景一：虚拟机单网卡（仅本机上网）

目的：让本机通过虚拟机中的爱快路由上网。

网卡配置：

  • 网卡0：桥接到真实网卡（作为WAN口）
  • 网卡1（VMnet1）：仅主机模式（作为LAN口）

配置步骤：

  1. 虚拟机添加一块网卡，总共两块
  2. 在虚拟网络编辑器中设置桥接
  3. 对VMnet1设置IP：192.168.1.2，子网掩码 255.255.255.0，网关 192.168.1.1
  4. 桥接的真实网卡取消IPv4设置
  5. 开机进入爱快控制台，绑定WAN口
  6. 浏览器访问 192.168.1.1 进入管理界面

4.3 场景二：虚拟机单网卡（本机+带机上网）

目的：本机和局域网内其他设备都通过虚拟机中的爱快上网。

网卡配置：

  • 网卡0（VMnet0）：桥接真实网卡（作为WAN口/LAN2口）
  • 网卡1（VMnet8）：NAT模式（本机上网）
  • 网卡8（VMnet8）：仅主机模式（作为LAN口）

额外步骤：

  1. 登录Web界面，绑定WAN口并配置外网接入方式
  2. 绑定LAN2口用于带机上网，LAN2的IP设为不同网段（如 192.168.10.1）
  3. 由于只有一个真实网口，需要外接二层交换机，将外网线和终端设备都接在交换机上

4.4 场景三：虚拟机多网卡（本机+带机上网）

目的：有多个真实网卡时，通过虚拟机带其他设备上网，实现更灵活的流量管控。

网卡配置：

  • 网卡0：桥接第一个真实网卡
  • 网卡1：桥接第二个真实网卡
  • 网卡8（VMnet8）：仅主机模式（管理用）

配置要点：

  1. 虚拟机总共需要三块网卡
  2. 仅主机模式的网卡用于本机管理，设置同网段IP
  3. 两块桥接网卡分别作为WAN口和LAN口
  4. 桥接的真实网卡取消IPv4设置

建议：不熟悉虚拟机的用户不建议使用虚拟机方式，推荐直接使用物理机安装，或请有虚拟机经验的网络维护人员协助。

五、云主机安装

爱快支持在阿里云、腾讯云等云主机上安装，主要用于SD-WAN组网场景。

安装步骤

前提：先在云主机上安装一个Ubuntu系统。

操作命令：

# 步骤1：下载ISO安装包
# 32位系统：
wget https://www.ikuai8.com/download.php?n=/3.x/iso/iKuai8_x32_3.5.11_Build202112031159.iso -O ikuai8.iso

# 64位系统：
wget https://www.ikuai8.com/download.php?n=/3.x/iso/iKuai8_x64_3.5.11_Build202112031159.iso -O ikuai8.iso

# 步骤2：挂载ISO镜像
sudo mount ikuai8.iso /mnt

# 步骤3：复制ISO镜像启动文件
sudo cp -rpf /mnt/boot /

# 步骤4：重启系统
sudo reboot

重启后操作：

  1. 在腾讯云/阿里云控制台打开VNC界面（登录 -> 远程登录 -> VNC登录）
  2. 在VNC中正常进行ISO安装爱快系统
  3. 安装完成后进入控制台，选择”其他选项” -> “开启外网访问WEB”
  4. 通过外网IP访问爱快Web管理界面

六、安装方式对比总结

安装方式 适用场景 难度 性能
ISO U盘引导 实体机安装（最常用） 简单 最优
UEFI引导 新硬件实体机 简单 最优
IMG写盘 快速部署 中等 最优
GHOST恢复 运维人员批量部署 中等 最优
ISO光盘 老硬件 简单 最优
VMware虚拟机 测试学习/单机多功能 较高 较好
云主机 SD-WAN组网 中等 取决于云配置

七、常见问题

Q1：安装后找不到管理界面？

确认电脑IP和爱快LAN口在同一网段。爱快默认LAN口地址是 192.168.1.1，将电脑设置为 192.168.1.x 网段即可访问。

Q2：多块硬盘怎么办？

爱快只能安装在单块磁盘上，安装前请断开其他磁盘，或确认安装目标正确。

Q3：虚拟机网卡性能差？

VMware默认可能使用AMD网卡类型，建议修改为e1000类型（编辑.vmx文件）。

Q4：忘记Web管理密码怎么办？

可以通过连接显示器键盘进入控制台，选择重置密码选项。

Q5：官方技术支持

  • 客服热线：400-877-3227（全年9:00-21:00，法定节假日除外）
  • 官方论坛：爱快社区
  • 官方微信/APP：搜索”爱快”
