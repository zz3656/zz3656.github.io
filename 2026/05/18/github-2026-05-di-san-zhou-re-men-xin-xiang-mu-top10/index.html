前言

2026年5月第三周（5月16日～5月23日），GitHub 新建仓库中 AI Coding Agent 和 AI 文本处理方向最为活跃。本周亮点：Perplexity 开源 Go 生态包扫描工具、专为小模型优化的 AI 编程工具亮相、Mac 视频动态壁纸引擎登场。本文整理本周 Stars 增长最快的新建仓库 Top 10。

一、本周新增 Stars Top 10

🥇 第1名：FoundZiGu/GuJumpgate — 游戏跳转网络中间件

仓库地址： https://github.com/FoundZiGu/GuJumpgate
语言： JavaScript
Stars：2,046 | Forks：588

GuJumpgate 是一个高性能游戏跳转（Jumpgate）网络中间件，使用 JavaScript/TypeScript 实现，专注于分布式游戏服务器间的通信与玩家分流。支持高并发连接管理，适合构建 MMO、游戏网关等场景的基础设施。

🥈 第2名：thananon/9arm-skills — Shell 知识管理技能集

仓库地址： https://github.com/thananon/9arm-skills
语言： Shell
Stars：1,856 | Forks：244

面向开发者效率提升的 Shell 脚本技能集，收录日常开发中高频使用的命令行技巧、自动化脚本和系统管理方案。纯 Shell 实现，零依赖，适合 Linux 服务器运维和开发环境快速配置。

🥉 第3名：Doorman11991/smallcode — 专为小模型优化的 AI 编程工具

仓库地址： https://github.com/Doorman11991/smallcode
语言： JavaScript
Stars：1,307 | Forks：95

Smallcode 是一款针对小型 LLM（4B 级活跃模型）优化的 AI 编码助手，在 87% 基准测试得分下保持极低资源消耗。适合本地开发环境和低配置机器使用，为边缘 AI 编程提供轻量解决方案。

4. datawhalechina/Agent-Learning-Hub — AI Agent 学习路线与资料库

仓库地址： https://github.com/datawhalechina/Agent-Learning-Hub
语言： HTML
Stars：1,210 | Forks：127

Datawhale 出品的 AI Agent 系统化学习路线，汇集学习资料、实践项目和研究论文。涵盖 Agent 架构、工具调用、记忆机制、多 Agent 协作等核心主题，是入门和进阶 Agent 开发的一站式资源库。

5. perplexityai/bumblebee — 包与扩展的只读端点扫描器

仓库地址： https://github.com/perplexityai/bumblebee
语言： Go
Stars：1,157 | Forks：91

Perplexity 开源的 Go 语言只读端点扫描工具，用于在磁盘上扫描已安装的包、扩展及其开发状态信息。适用于依赖审计、供应链安全检测和开发环境盘点等场景。标签：golang, package-inventory, supply-chain-security

6. sapientinc/HRM-Text — 基于 HRM 架构的 1B 文本生成模型

仓库地址： https://github.com/sapientinc/HRM-Text
语言： Python
Stars：675 | Forks：61

HRM-Text 是一个基于分层推理模型（Hierarchical Reasoning Model）架构的 1B 参数文本生成模型，强化了复杂上下文的推理与生成能力。支持多轮对话和长文本续写，适用于资源受限环境下的本地 AI 应用。标签：hierarchical-reasoning-model, hrm, large-language-models

7. kageroumado/phosphene — macOS Tahoe 视频壁纸引擎

仓库地址： https://github.com/kageroumado/phosphene
语言： Swift
Stars：625 | Forks：17

Phosphene 是一款 macOS Tahoe 原生视频壁纸引擎，支持将视频文件直接设为桌面壁纸。轻量原生实现，低内存占用，适合个性化桌面定制。

8. lynote-ai/humanize-text — 开源 AI 文本人性化工具

仓库地址： https://github.com/lynote-ai/humanize-text
语言： Python
Stars：549 | Forks：41

Lynote AI 出品的开源 AI 文本”人性化”工具，可将 AI 生成的内容转换为更自然、更难被检测的文本。支持多种风格迁移，适用于内容创作、SEO 文章和社交媒体运营场景。标签：ai-detection, ai-humanize, ai-humanizer

9. WUBING2023/PaperSpine — 论文驱动的 Codex 技能学习工具

仓库地址： https://github.com/WUBING2023/PaperSpine
语言： Python
Stars：514 | Forks：19

PaperSpine 是一款以强学术论文为驱动的 Codex 技能学习工具，帮助开发者通过阅读高质量学术成果来提升编程与研究能力。将论文阅读与 Codex 技能训练结合，适合 AI 研究者和学术开发者。

10. LiuMengxuan04/shushu-internship-tool — 简历面试转化工具

仓库地址： https://github.com/LiuMengxuan04/shushu-internship-tool
语言： Python
Stars：463 | Forks：18

把岗位描述变项目、把项目变简历、把简历变面试——这款工具帮助求职者系统化梳理实习/工作经历，将日常实践转化为结构化简历和面试答案，提升求职效率。标签：ai, ai-agents, skills

二、趋势分析

1. 小模型专用工具成新风口
Doorman11991/smallcode 和 perplexityai/bumblebee 同期上榜，分别从编程辅助和包扫描切入小模型生态，印证”大模型卷参数、小模型卷场景”的趋势正在深化。

2. AI Agent 基础设施持续丰富
Agent-Learning-Hub 提供学习路线、shushu-internship-tool 覆盖求职场景、PaperSpine 专注学术研究，AI Agent 正向教育、求职等更广泛的垂直场景延伸。

3. AI 文本处理赛道热度不减
humanize-text 和 HRM-Text 分别从”文本风格”和”推理生成”两个维度切入，AI 文本处理的商业化路径逐渐清晰，内容创作和本地部署需求同步增长。

4. 开发者效率工具百花齐放
9arm-skills（Shell 脚本）、phosphene（桌面定制）和 GuJumpgate（游戏中间件）横跨运维、个性化与游戏领域，表明开源生态正从”AI 热潮”向”全栈工具链”均衡发展。
