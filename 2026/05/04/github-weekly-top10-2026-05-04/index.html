GitHub 每周热门开源项目 TOP 10（2026年05月04日）

本周 GitHub 开源圈火力全开：AI 编码工具持续爆发，设计类项目异军突起，安全研究也引发广泛关注。以下是本周两个榜单的 TOP 10。

🔥 榜单一：本周热门新项目 TOP 10

统计周期：2026-04-27 ~ 2026-05-04（本周新创建的项目，按 Star 数排序）

1. open-design ⭐ 18.6k

本地优先的开源设计工具，对标 Anthropic Claude Design。内置 19 种 Skill 和 71 套品牌级设计系统，可生成网页、桌面、移动端原型，还能做幻灯片、图片和视频导出。兼容 Claude Code、Codex、Cursor、Hermes 等主流 AI Agent，是目前 AI 设计赛道最热门的项目。

TypeScript | 创建于 2026-04-28

https://github.com/nexu-io/open-design

2. copy-fail-CVE-2026-31431 ⭐ 3.0k

针对 CVE-2026-31431 漏洞的安全研究 PoC 代码。该漏洞影响范围广泛，项目提供了漏洞复现和分析的方法，在安全社区引发了大量关注和讨论，适合安全研究人员参考学习。

Python | 创建于 2026-04-29

https://github.com/theori-io/copy-fail-CVE-2026-31431

3. mike ⭐ 1.6k

开源 AI 法律平台，利用人工智能为法律行业提供辅助工具。可以帮助律师和法律工作者进行法律文书分析、案例检索、合规检查等工作，降低法律服务的门槛。

TypeScript | 创建于 2026-04-29

https://github.com/willchen96/mike

4. whatcable ⭐ 1.4k

macOS 菜单栏小工具，用大白话告诉你每根插在 Mac 上的 USB-C 线到底支持什么功能——充电多少瓦、能不能传数据、支不支持视频输出等。解决了”抽屉里一堆线却不知道哪根能用”的刚需痛点。

Swift | 创建于 2026-05-01

https://github.com/darrylmorley/whatcable

5. codex-plusplus ⭐ 853

Codex++ 是 OpenAI Codex 桌面应用的增强调整系统，提供额外的自定义和扩展功能，让 Codex 用起来更顺手、更强大。适合已经在使用 Codex 但觉得默认功能不够用的开发者。

TypeScript | 创建于 2026-04-28

https://github.com/b-nnett/codex-plusplus

6. dictionary-of-ai-coding ⭐ 822

AI 编程术语词典，用通俗的大白话解释 AI 编码领域的专业行话。比如什么是 RAG、什么是 Agent、什么是 Function Call，每个词条都配有简明解释。对于刚入门 AI 编程的开发者来说是非常友好的参考工具。

TypeScript | 创建于 2026-05-01

https://github.com/mattpocock/dictionary-of-ai-coding

7. NTSB_FOIA_MU5735 ⭐ 793

MU5735 航班 NTSB（美国国家运输安全委员会）FOIA（信息自由法）请求数据的公开仓库，汇集了与该航班事故相关的公开文件和资料。

N/A | 创建于 2026-04-30

https://github.com/wrongly-cuddly-obsession/NTSB_FOIA_MU5735

8. dbx ⭐ 770

仅 15MB 的轻量级跨平台数据库客户端，一个工具连接所有主流数据库——MySQL、PostgreSQL、SQLite、Redis、MongoDB、DuckDB、ClickHouse、SQL Server 等。体积小、启动快，适合不想安装多个客户端的开发者。

Vue | 创建于 2026-04-29

https://github.com/t8y2/dbx

9. chromex ⭐ 767

基于 Codex 的 Chrome 浏览器侧边栏 AI 助手，能理解当前页面上下文、管理标签页、处理语音指令和图片工作流。相当于在浏览器里内置了一个懂你页面的 AI 搭档。

TypeScript | 创建于 2026-04-28

https://github.com/GENEXIS-AI/chromex

10. SoftwareCopyright-Skill ⭐ 665

中国软件著作权申请材料自动生成器。只需指向本地项目目录，就能自动读取代码并生成全套 .docx 软著申请材料，全开源免费，再也不用花钱找代理办软著了。对国内独立开发者和团队来说非常实用。

Python | 创建于 2026-04-29

https://github.com/Fokkyp/SoftwareCopyright-Skill

🚀 榜单二：Star 上升最快 TOP 10（近30天新项目）

统计周期：2026-04-04 ~ 2026-05-04（近一个月内创建，按 Star 增速排序）

1. mempalace ⭐ 50.9k

开源 AI 记忆系统，号称基准测试表现最佳且完全免费。为 AI Agent 提供持久化记忆能力，让 Agent 能记住过去的对话和上下文，不再每次对话都”失忆”。发布一个月即突破 5 万 Star，是目前 AI 基础设施层最火的项目。

Python | 创建于 2026-04-05

https://github.com/MemPalace/mempalace

2. open-design ⭐ 18.6k

同榜单一冠军，一周冲上 1.8 万 Star。AI + 设计赛道的热度可见一斑。

TypeScript | 创建于 2026-04-28

https://github.com/nexu-io/open-design

3. nuwa-skill ⭐ 17.0k

女娲 Skill——蒸馏任何人的思维方式。能把一个人的心智模型、决策习惯、表达风格提取出来变成可复用的 Skill。概念非常新颖，相当于给 AI 装上了某个人的”思考方式”，引发了广泛讨论。

Python | 创建于 2026-04-05

https://github.com/alchaincyf/nuwa-skill

4. gbrain ⭐ 12.9k

YC 掌门人 Garry Tan 开源的个人 OpenClaw/Hermes Agent Brain 配置。分享了他如何使用 AI Agent 的完整思路和工作流，包含他的 Agent 配置哲学和实战经验。名人效应 + 实用内容，双重加持。

TypeScript | 创建于 2026-04-05

https://github.com/garrytan/gbrain

5. awesome-gpt-image-2-API-and-Prompts ⭐ 11.8k

GPT-Image-2 的 API 使用和 Prompt 合集。收集了大量图像生成的优质 Prompt 模板和 API 调用示例，是做 AI 图像生成应用的实用参考手册。

Python | 创建于 2026-04-18

https://github.com/EvoLinkAI/awesome-gpt-image-2-API-and-Prompts

6. huashu-design ⭐ 11.6k

华语设计 Skill——专为 Claude Code 打造的 HTML 原生设计工具。支持高保真原型生成、幻灯片制作、动画效果，还能导出 MP4 视频。内置 20 条设计哲学和 5 维评审体系，中文开发者做设计的好帮手。

HTML | 创建于 2026-04-19

https://github.com/alchaincyf/huashu-design

7. OpenMythos ⭐ 11.5k

Claude Mythos 架构的理论重建项目。基于公开的研究文献，从第一性原理出发，试图还原 Claude 背后的架构设计思路。属于 AI 研究类项目，适合对大模型架构感兴趣的研究者。

Python | 创建于 2026-04-18

https://github.com/kyegomez/OpenMythos

8. design.md ⭐ 11.3k

Google 出品的 DESIGN.md 规范。定义了一种格式，用来向 AI 编码 Agent 描述项目的视觉身份和设计系统，让 Agent 在写代码时能保持一致的设计风格。相当于给 AI 立了一份”设计规范守则”。

TypeScript | 创建于 2026-04-10

https://github.com/google-labs-code/design.md

9. browser-harness ⭐ 9.9k

浏览器自动化线束工具，核心卖点是”自愈能力”——当页面结构变化导致脚本失败时，它能自动修复定位并继续执行。让 LLM 可以稳定地完成任何浏览器操作任务，大大提升了浏览器自动化的可靠性。

Python | 创建于 2026-04-17

https://github.com/browser-use/browser-harness

10. obscura ⭐ 9.8k

面向 AI Agent 和网页爬虫的无头浏览器，用 Rust 编写。主打高性能和反检测能力，能让 AI Agent 更稳定、更快速地浏览网页和抓取数据，不容易被目标网站识别为机器人。

Rust | 创建于 2026-04-13

https://github.com/h4ckf0r0day/obscura

  • 设计 × AI：open-design、huashu-design、design.md，设计赛道成为新热点
