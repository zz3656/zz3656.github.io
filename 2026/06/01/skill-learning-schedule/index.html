Hermes Agent 技能学习计划（6月1日-6月5日）

目标：在6月5日前完成全部 92个 技能的学习，并通过 Hexo 博客发布每日详细学习报告。

总体概览

天数 日期 分类 技能数 主要内容
第1天 6月1日 Creative（创意） 20 图像生成、设计系统、ASCII艺术、视频制作、排版
第2天 6月2日 autonomous-ai-agents + data-science + devops 11 编程代理、数据分析、无盘服务器、看板工作流
第3天 6月3日 dogfood + email + gaming + github + mcp 14 质量测试、邮件、游戏服务器、GitHub操作、MCP协议
第4天 6月4日 media + mlops + note-taking + productivity 24 音乐/GIF/视频、HuggingFace、模型服务、文档处理
第5天 6月5日 project + red-teaming + research + smart-home + social-media + software-development + yuanbao 23 项目管理、红队对抗、学术研究、智能家居、社交媒体、开发工具

第1天（6月1日）✅ Creative（20个技能）

图形与图表生成

技能 功能 核心特点
architecture-diagram 生成深色主题SVG架构图/云图/基础设施图 无API密钥，离线可用，输出HTML文件
excalidraw 创建手绘风格的Excalidraw JSON图表 可拖放到excalidraw.com编辑，无需账号
baoyu-infographic 21种布局 × 21种风格的信息图生成 提供组合建议如 linear-progression + ikea-manual
sketch 生成2-3个HTML设计变体进行比较 从上下文出发，定义设计系统后产出变体

ASCII艺术与文本布局

技能 功能 核心特点
ascii-art 终端生成ASCII艺术（pyfiglet/cowsay/boxes） 支持571种字体，二维码/天气图可用curl生成
ascii-video 视频/音频转彩色ASCII字符MP4/GIF 使用自适应色调映射保持字符可见性
pretext DOM-free文本布局（@chenglou/pretext） 文本围绕移动障碍物流动，60fps无重排

AI图像与视频生成

技能 功能 核心特点
baoyu-article-illustrator 为文章生成配图（类型×风格×调色板） 三维度：Type（6种）+ Style（6种）+ Palette（3种）
baoyu-comic 知识漫画创作（教育/传记/教程） 艺术风格：ligne-claire、manga、realistic等
comfyui 通过ComfyUI生成图片/视频/音频 两层架构：comfy-cli（安装）+ REST/WS API（执行）
pixel-art 图片转复古像素艺术 + 动画 14种预设：arcade(16色)、nes(54色)、gameboy(4色)等
manim-video 3Blue1Brown风格数学动画 需要LaTeX渲染公式，先几何后代数，动效分层

设计系统

技能 功能 核心特点
claude-design 一次性HTML设计原型（着陆页/原型/幻灯片） 从上下文出发，设计系统先行，避免AI设计垃圾
design-md 编写Google DESIGN.md令牌规范文件 npx @google/design.md lint/export，WCAG对比度验证
popular-web-designs 54个现成设计系统（Stripe/Linear/Vercel等） 选择模板后可获取完整CSS/颜色值

创意工具

技能 功能 核心特点
creative-ideation 约束驱动项目灵感生成 约束+方向=创造力，如”解决自己的痛点”
humanizer 去除AI写作痕迹（29种模式） 5大类：内容/语言/风格/沟通/填充词，添加灵魂技巧
p5js 浏览器创意编程/生成艺术 禁用FES、种子随机、HSB颜色、createGraphics分层
songwriting-and-ai-music 歌曲创作 + Suno AI音乐提示词 结构ABABCB，押韵类型，动态”耳语→咆哮→耳语”
touchdesigner-mcp 通过MCP控制TouchDesigner实时视觉 36个原生工具，音频反应GLSL经验证配方

第2天（6月2日）autonomous-ai-agents + data-science + devops（11个技能）

autonomous-ai-agents（5个）：编程代理

技能 功能 核心命令/特点
claude-code 委托编码给Claude Code CLI -p 打印模式（首选），--max-turns 防无限循环，--resume 恢复会话
codex 委托编码给OpenAI Codex CLI exec "prompt" 一次性执行，--full-auto 自动审批变更
opencode 委托编码给OpenCode CLI opencode run 单次任务，opencode 交互模式（需pty）
hermes-agent 配置/扩展/贡献Hermes Agent 用于配置hermes本身
kanban-codex-lane Kanban worker用Codex作为隔离实现lane Hermes保留任务所有权，Codex执行实现

关键区别：

  • claude-code：Anthropic出品，print模式最干净，支持子代理、MCP、hooks
  • codex：OpenAI出品，必须在git仓库中运行，支持worktree并行
  • opencode：开源，provider-agnostic，支持session管理和stats

data-science（1个）

技能 功能 核心特点
jupyter-live-kernel 有状态Python REPL（通过live Jupyter kernel） 状态跨执行持久，变量/对象存活，适合数据分析

对比：execute_code是无状态单次脚本，jupyter-live-kernel适合迭代探索。

devops（4个）：运维与部署

技能 功能 核心特点
diskless-server 商业级无盘服务器设计/部署/运维 覆盖iSCSI+LIO+PXE/iPXE+dm-cache多级缓存架构
kanban-orchestrator 看板协调者分解playbook + 反诱惑规则 “不自己做工作”规则，生命周期自动注入
kanban-worker Hermes Kanban worker的坑/案例/边缘情况 具体场景的详细指导
webhook-subscriptions Webhook订阅：事件驱动agent运行 事件驱动架构

学习方法总结

每日流程

1. 使用 skill_view(name) 加载当天所有技能
2. 阅读 SKILL.md 了解：
   - 功能描述（这个技能做什么）
   - 核心命令（怎么用）
   - 核心特点（与其他工具的区别）
   - 适用场景（什么时候用）
3. 总结每个技能的特点和应用场景
4. 生成博客文章（中文，详细）
5. hexo generate && hexo deploy 发布

技能命名规范

  • 文章正文：全中文，流畅易读
  • 技能名：保留英文，如 claude-code、jupyter-live-kernel
  • 命令/代码：保留英文原文

质量标准

每个技能的介绍包含：

  • 一句话功能描述
  • 核心命令或使用方式
  • 与同类工具的区别（如果适用）
  • 适用场景
